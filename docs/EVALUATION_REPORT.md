## 二、项目定位

**AutoRecon 是一个高性能异步信息收集框架**，专注于：

- 🔍 **资产发现** - 子域名、端口、服务识别
- 🔒 **漏洞检测** - 敏感文件泄露、SQL注入、XSS
- 📊 **安全评估** - 压力测试、抗压等级评估
- 📋 **报告生成** - HTML/PDF 专业报告

**适用场景**：
- 渗透测试前期信息收集
- 资产安全评估
- 漏洞扫描与验证
- 网站性能压力测试

